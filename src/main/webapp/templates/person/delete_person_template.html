Delete a person: {{person_name}}

Deleting a person will remove them from the database. However, some data will remain:

  1. Their data will be stored in a "trash" folder
  2. Audits regarding actions will remain for the user who took those actions
  3. The log will contain some information about the person for a while
  4. The audits for a person will remain in the audit directories
  5. Their photos will be deleted. However, if the photo has been copied to another person, the copy will remain
  6. In the relationships section of other persons who link to this person, the dead-link will show in red
